# Pokedex App

The Pokedex App is used to see stats on Gen 1 Pokemon and choose your favorites.


## Technologies used

Tech stacks used in this app include:

-HTML

-CSS

-JavaScript

-JQuery

-Bootstrap

In this app, I also learned about Ajax and how to incorporate APIs.
